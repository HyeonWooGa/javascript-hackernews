# Hacker News Client

## 개요

- 대표적인 개발자 소셜 뉴스네트워크 서비스인 해커 뉴스의 API를 사용해 클라이언트를 만듭니다.
- Vanilla JS만 사용하여 개발 후에 TypeScript로 마이그레이션 해줍니다.

## Tech Requirement (Tech Stack)

- Vanilla JS (DOM, EventListner)
- Tailwind (Styles)
- Parcel (Server)
- Ajax (API)
